# Zinc Design System

Design system for Explore Joplin (explorejoplin.co).

Built on Untitled UI React + Tailwind CSS v4 + React Aria.

## Brand
- Primary: Graffiti (teal) — #15B79E
- Secondary: Blurple (purple) — #683DEE
- Neutral: Zinc — #09090B dark, #FAFAFA light
- Font: General Sans

## Components
Button · Input · Textarea · Tag · Slider · Badge · Card · StepDots · PageShell · ToolHeader · ToolFooter · Logo (Hologram / White / Black)

## Usage in products
Copy src/components, src/tokens, src/utils, and public/fonts into your product repo.
Import from the component files directly.

## Dev
npm run dev
